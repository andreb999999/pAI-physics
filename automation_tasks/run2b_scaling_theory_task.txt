Theory run for Muon scaling rules.

This stage builds on completed theory1 (implicit regularization) and experiments (fair comparison).

Advisor guidance:
"Look into the question: does a scaling rule exist for Muon?
Scaling rule means scaling between batch size and step size.
E.g., for an increase in batch size, how much of an increase in step size must occur to get the same results (e.g., to get the same 1-step decrease in loss)?
Critical batch size?"

- Search for: "gradient noise scale" (McCandlish et al.), "large batch training" scaling laws, "sign gradient noise", "majority vote gradient".

4) Connect scaling results back to the implicit regularization findings from theory1.
