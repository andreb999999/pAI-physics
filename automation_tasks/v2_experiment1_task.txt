2) IMPLEMENT MUON OPTIMIZER:
   - Create a PyTorch Muon optimizer class compatible with NanoGPT's training loop.
   - Muon update rule:
     a) Compute gradient G of loss w.r.t. weight matrix W.
     b) Compute the polar factor of G via Newton-Schulz iterations:
        Initialize X_0 = G / ||G||_F, then iterate 5 times:
        X_{k+1} = 1.5 * X_k - 0.5 * X_k @ X_k^T @ X_k
     c) Apply momentum: m_t = beta * m_{t-1} + X_k (the polar factor)
     d) Update: W_{t+1} = W_t - eta * m_t
   - Handle non-matrix parameters (biases, LayerNorm weights/biases, embeddings) with standard Adam/AdamW — Muon's polar decomposition is only meaningful for 2D weight matrices.
   - Ensure the implementation is numerically stable (handle zero gradients, small matrices).
