Experiment run for Muon implicit regularization empirical validation.

Objectives:
1) Design and execute experiments that directly test theoretical predictions about Muon's implicit L1 regularization.
2) Produce a fair-comparison methodology (fairness protocol) for optimizer evaluation.
3) Run all experiments locally on tractable surrogate tasks and save reproducible artifacts.

Experiment 1: Optimizer Comparison (Muon vs AdamW vs Adam)
- Task: Matrix factorization (A = UV^T) and logistic regression on synthetic data.
- Also include a small transformer language model if feasible (2-layer, 128-dim, synthetic vocab).
- Metrics: training loss, validation loss, generalization gap (train - val), convergence speed (steps to threshold).
- Fairness protocol: equal compute budget, equal hyperparameter search budget (at least 5 configs per optimizer), 5+ seeds per config, paired statistical tests.
- Report means, standard deviations, p-values for all pairwise comparisons.

Experiment 2: Weight Sparsity Analysis (CRITICAL - tests L1 regularization prediction)
- For each optimizer (Muon, AdamW, Adam), measure at convergence:
  a) L1 norm of final weights: ||W||_1
  b) L2 norm of final weights: ||W||_2
  c) L1/L2 ratio (sparsity proxy)
  d) Fraction of weights with |w_i| < threshold for thresholds {1e-4, 1e-3, 1e-2}
  e) Weight magnitude histogram/distribution statistics (mean, median, kurtosis)
- Track these metrics throughout training (every 100 steps), not just at convergence.
- If Muon has implicit L1 regularization, we expect: higher sparsity, lower L1 norm, more peaked distribution near zero.
- Compare Muon weights against AdamW + explicit L1 regularization (lambda = eta) weights. They should be similar if the theory is correct.

Experiment 3: Scale Invariance Test (tests sign-gradient scale invariance)
- Run Muon on loss L, 0.1*L, 10*L, 100*L.
- Record full parameter trajectory W_t for each.
- Verify trajectories are identical (or nearly so, within numerical precision).
- Run same test with AdamW; AdamW trajectories should differ across scales.
- Report: max trajectory deviation ||W_t^{scale} - W_t^{base}||_inf across all t.

Experiment 4: Regularization Strength Ablation (tests lambda ~ eta prediction)
- Run Muon with learning rates eta in {0.001, 0.003, 0.01, 0.03, 0.1}.
- For each eta, run AdamW with explicit L1 regularization at lambda = eta (same value).
- Compare final weight distributions: if theory is correct, Muon(eta) and AdamW+L1(lambda=eta) should produce similar sparsity patterns.
- Metric: KL divergence or Wasserstein distance between weight magnitude distributions.

Experiment 5: Gradient Norm and Training Dynamics
- Track per-step: gradient L2 norm, parameter L2 norm, effective learning rate, loss.
- Compare gradient norm stability across optimizers.
- Plot loss curves and gradient norm curves.
